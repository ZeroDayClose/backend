# Administration

Platform administration and configuration for ZeroDayClose.

---

## Overview

ZeroDayClose provides enterprise-grade administration features for managing users, configuring automation policies, and monitoring system health.

---

## Administration Areas

| Area | Description |
|------|-------------|
| [User Management](user-management.md) | Users, roles, permissions, and SSO |
| [Configuration](configuration.md) | System settings, policies, and thresholds |
| [Integrations Management](integrations-management.md) | Monitor and manage data connections |
| [APIs & Webhooks](apis-webhooks.md) | Programmatic access and extensibility |

---

## Quick Access

### User Management
- Add/remove users
- Assign roles and permissions
- Configure SSO (SAML/OAuth)
- Manage 2FA settings

### System Configuration
- Set approval thresholds
- Configure automation policies
- Define entity structure
- Map chart of accounts

### Monitoring
- Integration health status
- Sync schedules and history
- Error logs and alerts
- System performance
