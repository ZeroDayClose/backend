# User Interface

ZeroDayClose provides a modern, intuitive interface designed for finance professionals.

---

## Overview

The platform offers a web-based unified application optimized for:
- **Clarity and auditability** — Every number is traceable
- **Professional finance users** — High-density, data-rich interfaces
- **Hybrid AI/human workflows** — Seamless collaboration between AI and teams

## Key Principles

### 1. Drill-Through Everywhere

Every number in a dashboard can be clicked to see its source:
- Dashboard metric → Entity breakdown → Transaction list → Source document

### 2. AI Transparency

AI-generated items are clearly marked:
- Visual indicator (icon) showing AI-produced content
- Link to supporting rationale and computation trail
- Confidence score display

### 3. Contextual Actions

Actions appear where you need them:
- Approve/reject buttons on exception items
- Quick edit on line items
- One-click navigation to related data
